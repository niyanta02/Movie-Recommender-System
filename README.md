# Movie-Recommender-System

Build a content-based movie recommendation system
using movie metadata such as genres, overview,
keywords, cast, and director.

Workflow:
1. Load datasets
2. Merge movie and credits data
3. Clean and preprocess data
4. Perform feature engineering
5. Create combined textual features
6. Vectorize movie descriptions
7. Compute cosine similarity
8. Recommend similar movies
